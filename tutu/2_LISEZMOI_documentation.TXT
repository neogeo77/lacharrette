******************************
UTILISATION
******************************
Le site peut est entièrement administré par les menus qui le compose, SAUF :
	- pour créer une nouvelle charrette
	- pour supprimer une charrette


Pour créer une nouvelle charrette : dans la base de données (avec PhpMyAdmin par exemple) :
	- ajouter le trajet dans la table "trajet"
	- ajouter au moins une personne dans la table "personne"
	- ajouter une ligne dans la table "compteur", avec le code du nouveau trajet, et la date du jour - (moins) 9 jours (exemple, si nous sommes le 22/07/2009, mettre dans cette table la date 13/07/2009)
	- ajouter une ligne dans la table "compteur" avec le code du nouveau trajet, le code de la personne créée ci-dessus et nbreC=0, nbreP=0, reportNbreC=0, reportNbreP=0, nbreCDepart=NULL, nbrePDepart=NULL (ces compteurs devront être ajustés par la suite, depuis le site)
	- ajouter une ligne dans la table "appartenance"

Tout le reste (ajout des autres personnes etc...) peut ensuite être effectué directement depuis le site.
